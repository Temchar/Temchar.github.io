The Genuis Hour Project

My Driving Question

My driving question for this project was, How can I transform one of my previous drawings into an animation?.

Why I Choose This

I choose Animation because I know I want to be an artist of some form but currently I'm open to options.
For example, I may be a tattoo artist, or I may be an animator. So I have a wide range of options. So here was my thought process,
I have so many things I could be when I'm older how about I use this project to strike an option down so I can say I've tried it and
whether or not I liked it.

The Research

I looked at many tutorials to figure out what I would have to do. I looked for things that would explain how to animate and normally explain it very simply,
so that I may figure out how to use Adobe Character Animate.

The Problems

So This wasn't even the actual thing I had been working to make for the whole quarter.
See I made an animation but It doesn't work with this webpage
so I had to do some problem solving. I made this with a video creation
from images that was on my peronal laptop. Since it was my first time using
that I would say I did pretty well. But the reason why my animation i had
been working really hard on didn't work is because Adobe Media Encoder had some
issues and having 3 days left untill I had to turn in the project did not help.
The issues it had were the facts that it did not have half of the compatable
file extentions with a webpage on it. Adobe Media Encoder
also was not very easy to use and most of the time errors came up and I had no
idea how to fix them. So once I had gotten something to work and tried to
put that in my website it did not work. So I had to use this backup.
Which does not look very bad but is not very long so, it is hard to
tell whats suposed to be happening in it.

Overveiw

Overall, this was a huge challenge but then again I was diving into something new and I knew I was being a bit ambitious after the problems came in. This was supposed
to test out to see if I would like to major in this, and I think it was a success because I think if I were wanting to, I could now choose animating as a major because I now
understand it is going to be a lot of work and I may have many problems. If I were to choose this as a major, I would have to understand this and be okay with this. If I would choose this,
I most likely will not use this app to do this again. I will say that I think it is cool and all but, I found it a bit hard to use. I thought what I was going to use was Adobe Flash
which was the then replaced by Adobe Character Animate. Adobe Adobe Flash was a drawing frame-by-frame app and that was what I thought I was going to be using for the Genius Hour Project.
